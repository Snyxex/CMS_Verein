# 🏆 Vereins-CMS 2026 (Schulprojekt)

Dieses Repository enthält ein modulares Content-Management-System (CMS), das im Rahmen eines **Schulprojekts** entwickelt wurde. Ziel des Projekts war es, eine funktionale Web-Anwendung für Vereine zu erstellen, die sowohl eine öffentliche Präsenz als auch ein sicheres Verwaltungs-Backend bietet.

---

## 🏫 Projekt-Kontext
Dieses System wurde als Lernprojekt konzipiert, um die Grundlagen der Full-Stack-Webentwicklung zu demonstrieren. 
- **Fokus:** Saubere Trennung von Frontend und Backend (Separation of Concerns).
- **Lernziel:** Anwendung von Vanilla JavaScript, Web Components und asynchroner Kommunikation (Fetch) mit einem PHP-Backend.

---

## 🚀 Kern-Features

### 🌐 Öffentlicher Bereich
- **Dynamische Homepage:** Automatische Anzeige der Vereinsbeschreibung, aktueller News und des nächsten Termins direkt aus der Datenbank.
- **Wochenkalender:** Eine interaktive Web-Component zur Anzeige von Vereinsterminen inklusive Detail-Ansicht (Modals).
- **Mitglieder-Galerie:** Öffentliche Vorstellung des Teams mit Rollen und Beschreibungen.
- **Gästebuch:** Einfache Möglichkeit für Besucher, Feedback zu hinterlassen.

### 🔐 Admin-Dashboard
- **Zentrale Statistik:** Übersicht über Mitgliederzahlen, Events und Beiträge auf einen Blick.
- **News-Verwaltung:** Vollständiges Interface zum Erstellen, Bearbeiten und Löschen von Beiträgen mit Echtzeit-Suche.
- **Event-Management:** Planung von Terminen inklusive Standortdetails und Adressdaten.
- **Benutzer-Kontrolle:** Verwaltung von Administratoren und Vereinsmitgliedern in einer kombinierten Oberfläche.
- **Moderation:** Schnelle Löschfunktion für Gästebucheinträge zur Qualitätssicherung.

---

## 🛠 Technologie-Stack

| Bereich | Technologie |
| :--- | :--- |
| **Frontend** | HTML5, CSS3 (Modernes Flexbox/Grid), Vanilla JavaScript (ES6+) |
| **Komponenten** | Custom Web Components (Wiederverwendbare UI-Elemente) |
| **Backend** | PHP 8.x mit XAMPP |
| **Datenbank** | MySQL |
| **Datenfluss** | Asynchroner Datenaustausch via Fetch API (JSON) |
